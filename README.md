# movie_success_prediction
movie_success
Overview
This project analyzes movie data to understand how ratings, audience engagement, and sentiment influence box office success. It includes sentiment analysis on movie descriptions and a machine learning model to predict revenue.
Dataset source: IMDb
Tools & Technologies
Microsoft Excel – Data cleaning
Python – Data analysis
NLTK – NLP
VADER Sentiment Analyzer – Sentiment analysis
scikit-learn – Machine learning
Matplotlib, Seaborn – Visualization
Workflow
Data cleaning using Microsoft Excel
Sentiment analysis on movie overviews
Data visualization (charts & graphs)
Build Linear Regression model to predict revenue

Conclusion
The project shows that higher ratings, more votes, and positive sentiment generally lead to better box office performance.
